## 做个简单的数值运算

```python
a = 10
b = 20
c = a + b
print(c)
```

> python中没有变量的显示声明，但python不是弱类型的语言，
> 恰恰相反，python是强类型编程语言，只不过python中的变量类型是在运行时确定的

## Python中一些常用类型

- 数字
- 字符串
- 元组
- 列表
- 字典
- 集合

```python
# 字符串
s = 'Hello World'

# 数字
digital = 10

# 小数
dec = 10.5

# 元组
tup = ('Hello', 'world')

# 列表
lst = ['Hello', 'world']

# 字典
dst = {'Hello': 'world'}

# 集合
st = {'Hello', 'world'}
```
