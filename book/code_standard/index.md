## PEP8

> Style Guide for Python Code

https://www.python.org/dev/peps/pep-0008/

**代码排版**

- 缩进 4个空格
- 每行最大长度79
- 类和top-level函数之间空两行, 类中的方法之间空一行

**文档排版**

- 不要在一句import中多个库，比如import os, sys不推荐
