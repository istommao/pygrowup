# 时间日期

## time

```python
import time

# 时间戳
time.time()
```

## datetime

```python
from datetime import datetime

datetime.today()
```

## date

```python
from datetime import date

date.today()
```

## timedelta

```python
from datetime import datetime, timedelta

datetime.now() - timedelta(hours=1)
```

## calendar

```python
import calendar

print(calendar.month(2016, 6))
```
